About This Course

Build the web
from scratch.

HTML is the skeleton of every webpage on the internet. Before you can style or script anything, you need HTML — the language that gives the web its structure and meaning.

Designed by Babatunde Awoyemi at Techbase Consultant Services, Ibadan. 34 lessons taking you from absolute zero to confidently publishing your own web pages.

  • Headings, paragraphs, text formatting and links
  • Images, tables, lists and forms
  • Semantic HTML — header, nav, main, footer, article
  • Responsive design and layout techniques
  • Canvas, SVG, media and HTML5 Web APIs
