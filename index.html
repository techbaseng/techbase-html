About This Course

Build the web
from scratch.

HTML is the skeleton of every webpage on the internet. Before you can style or script anything, you need HTML — the language that gives the web its structure and meaning.

Designed by Babatunde Awoyemi at Techbase Consultant Services, Ibadan. These 34 lessons take you from absolute zero to confidently publishing your own web pages.

  • How browsers read and render HTML documents
  • Headings, text, links, images, and lists
  • Tables, forms, and user input elements
  • Semantic HTML — header, nav, main, footer
  • Responsive design and layout techniques
  • Canvas, SVG, media, and HTML5 Web APIs
