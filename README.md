## About This Course

HTML (HyperText Markup Language) is the foundation of every website on the internet. Before you can style a page or make it interactive, you need to understand HTML — the language that gives a webpage its structure and meaning.

In this course, you will build your first real web pages from scratch, learning how browsers read and display content using tags and elements.

**No prior experience needed. Just a browser and curiosity.**

---

## What You Will Learn

By the end of this course you will be able to:

- Understand how web browsers read and display HTML
- Create a properly structured HTML document
- Use headings, paragraphs, lists, links, and images
- Build tables and forms
- Write semantic HTML that is accessible and meaningful
- Publish a simple webpage on the internet

## How to Use This Course

1. **Start here** — read this page to understand the full picture
2. **Open Lesson 01** — click the link in the table above
3. **Read the lesson** — each file explains the concept with examples and code
4. **Try the exercises** — every lesson ends with hands-on practice tasks
5. **Move forward** — go to the next lesson whenever you are ready

> 💡 **Tip:** You do not need to install anything. Open [VS Code](https://code.visualstudio.com) or even Notepad, write your HTML, save as `.html`, and open the file in your browser to see it live.
